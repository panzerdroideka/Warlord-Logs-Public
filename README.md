# Warlord Combat Logs & Telemetry
Offentlig arkiv for ytelsesdata og kamp-logger fra Middle-earth.

Her finner du rådataene som beviser ytelsen og slagkraften du ser i mine videoer på YouTube.

📊 Hva du finner her:
/Showcase: Her ligger de nyeste loggene fra mine "Missions" og raid.

Combat Logs: Fullstendige utskrifter av kampforløp, inkludert de dypeste detaljene bak mine "Epic Conclusions".
